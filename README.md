# 🚀 Practicas "Redes y Sistemas distribuidos para control" (34396)
> Repositorio de prácticas para la asignatura Redes y Sistemas distribuidos para control (34396).

## 📝 Descripción
Este repositorio contiene las prácticas y tareas del curso de Redes y Sistemas distribuidos para control (34396), enfocadas en protocolos de comunicaciones TCP, UDP e IP.

---

## 📁 Estructura del Proyecto

Esta es la organización actual de las carpetas:

* **[P1](./P1):** Descripción de la 1º Práctica: Sincronización de hilos.
* **[P2](./P2):** Descripción de la 2º Práctica: Estructuras de comunicación Cliente-Servidor con UDP.
* **[P3](./P3):** Descripción de la 3º Práctica: Estructuras de comunicación Cliente-Servidor con TCP y trasferencia de datos con dirección específica.
* **[P3](./P4):** Descripción de la 4º Práctica: Comunicaciones Subscriptores-publicadores en MQTT.
* **[Tareas](./Tareas):** Contiene los ejercicios que no son prácticas y entregas adicionales.

## 📚 Librerías Utilizadas

### 📦 Biblioteca Estándar de Python (Nativas)
* **[threading](https://docs.python.org/3/library/threading.html):** Gestión de hilos y ejecución concurrente.
* **[time](https://docs.python.org/3/library/time.html):** Funciones relacionadas con el tiempo y retardos.
* **[random](https://docs.python.org/3/library/random.html):** Generación de números pseudoaleatorios.
* **[socket](https://docs.python.org/3/library/socket.html):** Interfaz de red de bajo nivel (comunicaciones TCP/UDP).
* **[datetime](https://docs.python.org/3/library/datetime.html):** Manipulación de fechas y horas.
* **[tkinter](https://docs.python.org/3/library/tkinter.html):** Interfaz gráfica de usuario (GUI) estándar de Python.
* **[pickle](https://docs.python.org/3/library/pickle.html):** Serialización y deserialización de objetos Python.
* **[uuid](https://docs.python.org/3/library/uuid.html):** Generación de identificadores únicos universales.
* **[json](https://docs.python.org/3/library/json.html):** Codificación y decodificación de datos en formato JSON.

### 🌐 Librerías Externas (Instalables via pip)
* **[Matplotlib](https://matplotlib.org/stable/index.html):** Biblioteca para la creación de visualizaciones estáticas, animadas e interactivas.
  * **pyplot:** Interfaz basada en estados para gráficos.
  * **animation:** Herramientas para la creación de animaciones dentro de los gráficos.
* **[paho.mqtt](https://pypi.org/project/paho-mqtt/):** Intermediario para aplicaciones a un broker MQTT para publicar y suscribirse a mensajes.
  * **client:** Módulo utilizado para gestionar la conexión, publicación y suscripción a tópicos MQTT.
